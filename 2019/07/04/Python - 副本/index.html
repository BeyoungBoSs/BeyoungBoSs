一些方法

  1. .append('123')将元素 ‘123’ 放入列表末尾
  2. .insert(0.'123')将元素 ‘123’ 放入索引 0 处,其他元素右移
  3. del t[0]永久删除索引为 0 处的元素
  4. pop(0)临时删除（列表里不存在，在临时变量里）索引 0 处的元素，默认索引-1
  5. remove('123')临时根据值删除元素，只删除顺序第一个，无法多个
  6. sort()永久性排序，参数reverse=True为倒序，默认reverse=False正序car.sort(reverse=False)#方法
  7. sorted()临时性排序sorted(t)#函数
  8. reverse()反转列表
  9. len()表长

Lambda函数

  1. lambda允许用户快速定义单行函数，当然用户也可以按照典型的函数定义完成函数。

  2. lambda定义的是单行函数，如果需要复杂的函数，应该定义普通函数

  3. lambda参数列表可以包含多个参数，如lambda x,y:x+y

  4. lambda中的表达式不能含有命令，而且只限一条表达式
