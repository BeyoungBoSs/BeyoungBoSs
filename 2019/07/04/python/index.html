基础

变量的命名和使用

  1. 变量名只含 字母、数字、下划线 不能有空格
  2. 慎用小写字母‘i’和大写字母‘O’

    字符串

  3. 引号成对出现
  4. print('hello "world"')
  5. print('''hello "world"''')

    一些方法

  6. .title()：首字母大写
  7. .upper()全大写
  8. .lower()全小写
  9. .rstrip()删除字符串末尾的空白（但是保存在临时变量里，想改变原变量就要重新赋值回去）
  10. .lstrip()删除字符串首段的空白
  11. .strip()删除字符串两边的空白
